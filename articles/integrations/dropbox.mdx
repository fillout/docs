## What Dropbox is

[Dropbox](https://www.dropbox.com/) allows users to store documents, photos, videos, and more in your personal cloud storage space, eliminating the need for physical storage devices.

## Video tutorial

https://youtu.be/9btbb2SAtY0?feature=shared

## How it works

### 1. Link your Fillout form to Dropbox

Go to **Integrate** and click `Dropbox`.

![Screenshot2025 05 15at4 42 27PM Pn](/images/Screenshot2025-05-15at4.42.27PM.png)

<Check>
  **Important:** This integration is only applicable to **file uploads**. When creating your form, make sure you have at least one file upload field.
</Check>

Add a name for your connection, then click `Connect to Dropbox`. Enter your log-in details in the pop-up window that appears, then click `Allow`.

### 2. Map your form fields on **Fillout**

Choose which **Folder** you would like to store the files in. Under **upload fields**, `+ Add new` and select a the field from your form.

![Screenshot2025 02 17at11 17 43AM Pn](/images/Screenshot2025-02-17at11.17.43AM.png)

### 3. **Publish and share**

Once everything is set, click `Finish setup` on the upper right corner. That's it\! Your form's good to go. Simply `Publish`and share the link.

## Add a dynamic file path

You can set a dynamic file path to categorize file uploads into folders that are based on user input. Toggle the `Custom subfolder and file name?` switch, then select a **Folder path fragment**. Enter a name or click `+` to map a field in your form.

![Screenshot2025 02 17at11 18 52AM Pn](/images/Screenshot2025-02-17at11.18.52AM.png)

Now, when a respondent submits a form, their file upload will be stored in a folder under their Name. This means every respondent will have their own folder.

## Add subfolders

You can add a subfolder by typing `/` followed by the subfolder name of your choice.

![Screenshot2025 02 17at11 20 45AM Pn](/images/Screenshot2025-02-17at11.20.45AM.png)

## Enter a personalized filename prefix

You can also add a **Filename prefix**. Similar to the steps above, enter text or click `+` to reference a field.

![Screenshot2025 02 17at11 21 31AM Pn](/images/Screenshot2025-02-17at11.21.31AM.png)
