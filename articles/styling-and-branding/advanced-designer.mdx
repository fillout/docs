## How to open the advanced designer

Click `Theme` to open the **Form designer**. Then, scroll down on the left-hand panel, and click `Advanced designer`.

![Adr1 Web](/images/adr1.webp)

<Note>
  **Note:** This feature is available on the **Starter** plan and above (except for custom CSS, which is available for Pro users and above). Learn more about our pricing plans and advanced features [**here**](https://www.fillout.com/pricing).
</Note>

Our **Advanced designer** lets users:

- [Add custom CSS](https://www.fillout.com/help/custom-css)
- Edit the input style
- Change the shape of buttons
- Choose a different animation

![Adr2 Web](/images/adr2.webp)

You can also:

- Change the text alignment
- Edit the input width
- Change the padding or spacing between fields

![Adr3 Web](/images/adr3.webp)
