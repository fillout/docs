## Overview

Adding **custom code** into your Fillout forms can be useful to:

- Inject analytics tracking scripts like Google Tag Manager
- Add HTML widgets like Intercom to the page

If you’re looking to style your form, use the [custom CSS feature](/help/custom-css) instead.

<Note>
  Custom CSS is available on the **Business** and **Enterprise** plans. For more details on pricing, click [**here**](https://www.fillout.com/pricing).
</Note>
## How to add a custom code


<Check>
  **Important:** To run a custom code, you’ll first need to connect a [custom domain](/help/custom-domains). Custom code will only run on live forms (not in preview mode) on your custom domain.
</Check>

<Steps>
  <Step title="Go to the Settings tab on your form">
  Click `Custom code`.

  <Frame >
  <img src = "/images/advanced/image-17.png" />
  </Frame>
  </Step>
  <Step title="Type your code">
  Type your code in the `Custom code` field.

  <Frame>
  <img src= "/images/advanced/image-18.png" />
  </Frame>
  </Step>
  <Step title="Publish your form and try it out!">
  Press `Save` afterwards.

  <Frame>
  <img src= "/images/advanced/image-19.png" />
  </Frame>
  </Step>
</Steps>
